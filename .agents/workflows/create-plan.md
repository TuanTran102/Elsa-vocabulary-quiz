# Steps
1. Get content of the task (e.g., CM-13) from `.agents/tasks`.
2. Create a plan file in `.agents/plans/` named `[TASK_KEY]-plan.md` (e.g., `CM-13-plan.md`).
3. The plan MUST:
   - Use TDD
   - Be written in **ENGLISH**.
   - Detail the implementation steps (logic, tests, UI).
   - NOT include actual code implementation.
4. Ask the me if any requirements need clarification before finalizing the plan.
